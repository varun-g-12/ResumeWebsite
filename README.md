# Resume Website - Varun G

> A modern, interactive resume website showcasing professional experience as a GenAI Developer with Apple-inspired design and smooth scroll effects.

## About the Project

This is a single-page resume website designed to present my professional journey, technical skills, and project portfolio in an engaging and visually appealing way. The website features modern web design principles with smooth animations and responsive layouts that work seamlessly across all devices.

The design takes inspiration from Apple's minimalist aesthetic, featuring clean white backgrounds, elegant typography, and sophisticated scroll-based animations that create an immersive user experience.

## Features

- **Parallax Scrolling Effects** - Background and foreground elements move at different speeds to create depth
- **Fade-in Animations** - Content smoothly appears as you scroll through sections
- **Sticky Section Headers** - Headers pin in place while scrolling for better navigation
- **Responsive Design** - Mobile-first approach ensuring perfect display on all screen sizes
- **Smooth Scrolling** - Buttery-smooth transitions between sections
- **Interactive Project Showcase** - Grid layout highlighting 6 major GenAI projects
- **Professional Timeline** - Visual representation of work experience at Merck Group
- **Skills Visualization** - Categorized tags showcasing technical expertise
- **Contact Section** - Easy access to email, phone, LinkedIn, and Medium profiles
- **Downloadable Resume** - One-click PDF download option

## Tech Stack

### Frontend
- **HTML5** - Semantic markup for better accessibility and SEO
- **CSS3** - Modern styling with Grid, Flexbox, and custom properties
- **JavaScript (ES6+)** - Vanilla JavaScript for scroll effects and interactivity

### Libraries & Tools
- **AOS.js** (Animate On Scroll) - Declarative scroll animations with data attributes
- **Custom Parallax** - Hand-crafted parallax effects using Intersection Observer API
- **Font Awesome** - Icon library for social media links
- **Google Fonts** - Apple-style system fonts (SF Pro Display, SF Pro Text)

### Hosting
- **GitHub Pages** - Free, fast, and reliable static site hosting

## Design System

**Color Palette:**
- Primary Background: Pure White (#ffffff)
- Text Primary: Near-Black (#1d1d1f)
- Text Secondary: Gray (#6e6e73)
- Accent: Apple Blue (#0071e3)
- Card Backgrounds: Subtle Gray (#f5f5f7)

**Typography:**
- Headings: SF Pro Display (48px+)
- Body: SF Pro Text
- Line Height: 1.6 for optimal readability

**Animation Principles:**
- Duration: 0.6-0.8s for smooth feel
- Easing: Cubic Bezier curves for natural acceleration
- Subtlety: Gentle movements (20-50px) to avoid overwhelming users
